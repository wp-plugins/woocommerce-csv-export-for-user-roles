Quickly and easily export Woocommerce orders based on user roles.

== Description ==

This plugin will allow you to export orders in a date range, and give you a summary of the order information sorted by user group. Quickly and easily find out how many sales have been made by your wholesale users vs your retail users. (Or any other user groups you have created)

== Installation ==

This section describes how to install the plugin and get it working.

1. Download and extract 'wooexportcsv.zip'
2. Upload `wooexportcsv` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to WooCommerce -> Export Orders
5. Export your orders!
